# Employee Management System

This is a web-based Employee Management System built using **Java, JSP, Servlets, and Hibernate**.  
The main goal of this project is to manage employee details such as adding, updating, viewing, and deleting employees in an easy way.  
It also includes a simple login system for admin access.

---

Features

- Login and logout functionality using sessions  
- Add new employees with name, email, father name, department, and salary  
- View all employees in a responsive table  
- Filter employees by department (e.g., Web Developer, Software Developer, etc.)  
- Update or delete employee details  
- Automatically stores the date when an employee is created  
- Mobile-friendly interface  

---

## Technologies Used

- **Frontend:** HTML, CSS, JSP  
- **Backend:** Java Servlets, Hibernate ORM  
- **Database:** MySQL  
- **Server:** Apache Tomcat  
- **Build Tool:** Maven  
- **IDE Used:** Eclipse / IntelliJ IDEA
